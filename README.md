# Netflix Data Analysis

## Project Overview

This project performs an Exploratory Data Analysis (EDA) on the Netflix Movies and TV Shows dataset using Python. The objective is to analyze Netflix's content library and uncover trends in genres, countries, content growth, and ratings.

---

## Dataset

- Source: Kaggle
- Dataset: Netflix Movies and TV Shows

---

## Tools & Libraries

- Python
- Pandas
- Matplotlib

---

## Project Workflow

1. Data Loading
2. Data Exploration
3. Data Cleaning
4. Movies vs TV Shows Analysis
5. Top 10 Genres
6. Top 15 Countries
7. Content Growth Analysis
8. Movies Added Each Year
9. TV Shows Added Each Year
10. Content Ratings Analysis

---

## Key Findings

- Movies outnumber TV Shows in Netflix's catalog.
- International Movies are the most common genre.
- The United States contributes the highest number of titles.
- Netflix experienced rapid content growth between 2016 and 2019.
- TV-MA and TV-14 are the most common content ratings.
