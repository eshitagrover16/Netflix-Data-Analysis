# 🎬 Netflix Data Analysis

## 📌 Project Overview

This project performs an **Exploratory Data Analysis (EDA)** on the Netflix Movies and TV Shows dataset using **Python, Pandas, and Matplotlib**.

The objective is to analyze Netflix's content library, identify trends, and generate business insights through data cleaning, transformation, and visualization.

---

## 📂 Dataset

- **Dataset:** Netflix Movies and TV Shows
- **Source:** Kaggle
- **Format:** CSV

---

## 🎯 Project Objectives

- Analyze the distribution of Movies and TV Shows.
- Identify the most popular genres.
- Determine which countries contribute the most content.
- Analyze Netflix's content growth over the years.
- Compare the yearly addition of Movies and TV Shows.
- Examine the distribution of content ratings.

---

## 📊 Project Workflow

### 1. Data Loading
- Imported the dataset using Pandas.
- Loaded the dataset from Google Drive.

### 2. Data Exploration
- Examined the dataset structure.
- Checked data types.
- Identified missing values.
- Explored the dimensions of the dataset.

### 3. Data Cleaning
- Removed leading and trailing spaces.
- Converted the `date_added` column into datetime format.
- Created the `year_added` feature.
- Split multi-value columns (`listed_in` and `country`).
- Used `explode()` for detailed analysis.

### 4. Exploratory Data Analysis (EDA)

- Movies vs TV Shows
- Top 10 Genres
- Top 15 Countries
- Netflix Content Growth
- Movies Added Each Year
- TV Shows Added Each Year
- Distribution of Content Ratings

---

## 📈 Key Findings

- Movies significantly outnumber TV Shows in Netflix's catalog.
- International Movies are the most common genre.
- The United States contributes the highest number of titles.
- Netflix experienced rapid content growth between 2016 and 2019.
- 2019 recorded the highest number of content additions.
- TV-MA is the most common content rating, indicating a large amount of mature content.
